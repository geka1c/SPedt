Поиск Участника

 

Чтобы выдать заказ Участнику необходимо

Считать карту участника либо считать Штрих код с телефона из мобильного приложения

Либо

Найти участника в программе и выдать ему заказы.


Строка поиска может состоять из

·         Логина на сайте

·         номера телефона (телефон зарегистрированный на сайте. Поиск по номеру осуществляется если номер начинается с символов «+7» либо «8»)

·         ИД участника. (Код участника на сайте. Чтобы поиск происходил по ид необходимо установить флаг «По ИД»)

Для осуществления поиска вводим строку поиска в соответствующее поле и нажимаем Найти.
