Потерянные заказы

Заказы, отправленные из ЕРЦ и не поступившие в установленный срок на пункт выдачи считаются потерянными. На рабочем столе появляется уведомление о количестве потерянных заказов

 

При нажатии на уведомление можно увидеть расшифровку по конкретным заказам

Сгруппированных по транспортной накладной.

Необходимо выяснить причину недопоставки. И отписаться о решении вопроса по каждому потерянному заказу.

Для этого, двойным кликом на заказе открываем форму решения вопроса. Ставим галочку вопрос решен. В комментарии кратко описываем результат решения.

Чтобы отписаться сразу по нескольким заказам выделяем их с нажатой клавишей shift или ctrl

И нажимаем «Отписаться»

 

 

Если заказ приехал позже установленного срока, происходит автоматическая отписка.  Отписка об опоздавших заказах запускается в фоне каждые 2 часа.

Можно принудительно отписаться об опоздавших заказах нажав «Отметить полученные»

 

Заказ уходит из потерянных, только после отписки.

 

По завершения работы с потерянными заказами

Нажимаем провести и закрыть.
